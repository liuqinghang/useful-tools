# Useful Tools

## Playwright
- **Description**: A library to automate the Chromium, WebKit and Firefox browsers with a single API.
- **Documentation**: [Playwright Docs](https://playwright.dev/docs/intro)
- **GitHub Repository**: [Playwright GitHub](https://github.com/microsoft/playwright)

## Bandizip
- **Description**: A powerful and fast archiving tool.
- **Official Link**: [Bandizip Official](https://www.bandizip.com/)

## SumatraPDF
- **Description**: A lightweight PDF, eBook (ePub, Mobi), CHM, XPS, DjVu, AVI, and Flash viewer.
- **Official Link**: [SumatraPDF Official](https://www.sumatrapdfreader.org/free-pdf-reader.html)

## Everything
- **Description**: A search engine for Windows that locates files and folders by name instantly.
- **Official Link**: [Everything Official](https://www.voidtools.com/)

## WizTree
- **Description**: A free tool that scans and displays all the files and folders on your hard drive in seconds.
- **Official Link**: [WizTree Official](https://wiztreefree.com/)
